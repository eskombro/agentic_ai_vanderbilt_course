# AI Agents — Coursera exercises

This Git repository stores practice code and exercises from **[AI Agents and Agentic AI with Python & Generative AI](https://www.coursera.org/learn/ai-agents-python/)** (Vanderbilt University, Coursera). The course teaches how to design and implement AI agents in Python from first principles—using ideas such as goals, actions, memory, and environment (GAME), programmatic prompting, agent loops, and LLM function calling—so you can build tool-using assistants that interact with files, APIs, and other systems. Along the way you gain skills in generative AI agents, prompt engineering, tool design, agentic workflows, and practical Python application development grounded in OpenAI-style APIs.

**Course:** [https://www.coursera.org/learn/ai-agents-python/](https://www.coursera.org/learn/ai-agents-python/)
